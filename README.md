## 🚀 What I'm Building

### **TicketBlitz** 🎟️
Currently engineering **TicketBlitz**, a highly scalable event booking platform designed to handle high-traffic surges and concurrent ticket reservations seamlessly. 

* **The Problem:** Preventing double-booking and system crashes during flash sales for high-demand events.
* **The Architecture:** * Powered by a **Spring Boot** microservices backend.
  * Utilizing **Apache Kafka** for robust, asynchronous message streaming and event-driven communication.
  * Leveraging **Redis** for sub-millisecond caching and distributed locking to ensure transaction integrity.
  * Containerized environments using **Docker** for consistent, scalable deployments.
